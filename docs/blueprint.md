# **App Name**: GlobalLib Discover

## Core Features:

- Book Browsing: Browse a catalog of books with details such as title, author, cover image, and description.
- Book Search: Allow users to search for books by title, author, or category.
- Detailed Book View: Provide a dedicated page for each book with a cover image, title, author, description, mood tags, affiliate link, and the full text.
- Category Filtering: Filter books by predefined categories.
- Mood Tag Filtering: Filter books based on mood tags to find books that match the reader's current mood.
- AI-Powered Book Recommendations: Suggests books to users based on their reading history and preferences, using a tool that selects relevant mood tags.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sophisticated and intellectual feel.
- Background color: Very light grey (#F0F0F0) for a clean, modern backdrop.
- Accent color: Muted lavender (#B886C9) for subtle highlights and interactive elements.
- Body font: 'Inter', a grotesque-style sans-serif, is used for all text on the page.
- Headline font: 'Space Grotesk', a sans-serif font, to highlight key section titles.
- Use 'lucide-react' icons for interactive elements like search, filter, and navigation.
- Employ a grid-based layout to present books in an organized and visually appealing manner.
- Incorporate subtle transition animations for loading and page navigation.
